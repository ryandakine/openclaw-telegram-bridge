## Running the Dashboard

```bash
# Start the dashboard server
./dashboard.sh

# Or with custom port
PORT=8080 ./dashboard.sh
```

Then open http://localhost:3456 (or your custom port) in your browser.

## Task Management

Tasks are stored in `.taskmaster/tasks/tasks.json`. The dashboard auto-refreshes every 30 seconds.
